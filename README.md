# 🎬 DramaBox — Інструкція

## Структура файлів

```
📁 ваш-репозиторій/
├── 📄 index.html        ← сам сайт (не чіпайте)
├── 📄 content.js        ← ВСІ ТЕКСТИ ТУТ (редагуйте цей файл)
├── 📁 videos/
│   ├── 🎥 episode1.mp4
│   ├── 🎥 episode2.mp4
│   └── 🎥 ...
└── 📁 images/           (необов'язково — обкладинки)
```

---

## ✏️ Як змінити текст серії

Відкрийте **content.js** → знайдіть потрібну серію → змініть:

```js
{
  title: "Нова назва серії",          // ← назва
  description: "Новий опис...",       // ← текст під назвою
  cliffhanger: "Інтрига в кінці...", // ← червоний рядок
  free: true,                         // ← true = безкоштовна, false = платна
}
```

---

## 🎥 Як додати нове відео

1. Завантажте відео в папку `videos/` у GitHub
   - Назвіть його `episode2.mp4`, `episode3.mp4` тощо

2. В `content.js` у потрібній серії вкажіть шлях:
   ```js
   video: "videos/episode2.mp4",
   ```

3. Якщо відео немає — залиште порожнім:
   ```js
   video: "",
   ```

---

## ➕ Як додати нову серію

В `content.js` знайдіть потрібний серіал і додайте новий блок в `episodes: [...]`:

```js
{
  id: 6,                              // ← номер (наступний по порядку)
  title: "Назва серії",
  description: "Опис того що відбувається...",
  cliffhanger: "Що буде далі...",
  video: "videos/episode6.mp4",       // ← або "" якщо немає
  poster: "",
  free: false,                        // ← false = платна
  duration: "2:00",
},
```

---

## ➕ Як додати новий серіал

В `content.js` у масив `serials: [...]` додайте:

```js
{
  id: 4,                              // ← наступний номер
  title: "Назва серіалу",
  genre: "Жанр • Жанр",
  emoji: "🔥",                        // ← емодзі для серіалу
  color: "#ff6b35",                   // ← колір (hex)
  gradient: ["#2a1000","#0a0500"],    // ← темний градієнт фону
  episodes: [
    {
      id: 1,
      title: "Перша серія",
      description: "Опис...",
      cliffhanger: "Інтрига...",
      video: "videos/new_ep1.mp4",
      poster: "",
      free: true,
      duration: "1:30",
    }
  ]
},
```

## 💰 Як змінити ціни

В `content.js` зверху:

```js
coinPrice: {
  small:  { coins: 10, price: "29 грн",  label: "Старт" },
  medium: { coins: 30, price: "69 грн",  label: "Популярний 🔥" },
  large:  { coins: 80, price: "149 грн", label: "Максимум" },
},
episodeCost: 5,  // ← скільки монет коштує одна серія
```

---

## 🌐 Як публікувати зміни на GitHub

1. Відкрийте **content.js** у репозиторії
2. Натисніть олівець ✏️ (Edit)
3. Змініть текст
4. Натисніть **Commit changes**
5. Сайт оновиться через ~1 хвилину

### Щоб завантажити нове відео:
1. Зайдіть у папку `videos/`
2. Натисніть **Add file → Upload files**
3. Перетягніть відео
4. Натисніть **Commit changes**
